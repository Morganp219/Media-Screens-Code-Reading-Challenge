About the Workshop

Responsive DevCamp is designed for developers who already know basic HTML and CSS, and want to become confident in building layouts that adapt to any screen size.

We’ll cover flexible grids, responsive typography, media queries, and layout patterns for multi-device experiences.

Learning Tracks

Track 1: Layout Basics

Level: Beginner

Understand how to structure HTML for responsive designs, using semantic tags and simple flexbox layouts.

Track 2: Media Queries

Level: Intermediate

Learn how to adapt layouts, font sizes, and navigation for small, medium and large screens using media queries.

Track 3: Advanced Patterns

Level: Advanced

Explore responsive cards, dashboard layouts, and multi-column designs that collapse gracefully on mobile.
